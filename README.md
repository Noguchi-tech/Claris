# Claris / クラリス

自分用のタスク、メモ、方針管理アプリです。まずは iPhone の Safari でホーム画面に追加して使える、ローカル保存の PWA として実装しています。

## 起動

```powershell
node server.mjs
```

起動後に `http://127.0.0.1:4173/` を開きます。

## 主な機能

- 下部タブ中央の「今日」
- タスク、メモ、方針、カレンダーの管理
- タスクごとの担当者設定
- 実施日と `DL` の分離
- タスク一覧の実施日順ソート
- 優先度枠の重複時の入れ替え UI
- カレンダーでのDL強調、優先度枠、期間ライン表示
- 走り書きメモから議題、決定事項、ネクストアクションへの整理
- 録音メモの端末内保存と録音時の文字起こし
- JSON / CSV / テキスト貼り付けインポート
- JSON エクスポート
- IndexedDB によるローカル保存

## 公開URLとマスタデータ

GitHub Pages の公開URLは静的配信なので、アプリから安全にマスタファイルへ直接書き戻すことはしません。直接書き込みを公開ページに持たせると、認証情報や書き込み口の扱いが難しくなります。

当面の運用は、設定の「マスター上書き用JSON」でiPhone側の最新状態を書き出し、PC側で `data/current-todo-tasks-2026-05-15.json` の更新に使う形です。パスキーなどで安全に同期する場合は、GitHub Pagesだけで完結させず、認証と書き込み権限を持つバックエンドを別途用意します。
