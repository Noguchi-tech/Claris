# Claris / クラリス

自分用のタスク、メモ、運営情報管理アプリです。まずは iPhone の Safari でホーム画面に追加して使える、ローカル保存の PWA として実装しています。

## 起動

```powershell
node server.mjs
```

起動後に `http://127.0.0.1:4173/` を開きます。

## 主な機能

- 下部タブ中央の「今日」
- カレンダー、今日、一覧の3タブ構成
- 今日上部の集計4項目の表示切り替え
- タスク、メモ、運営情報を同じ一覧画面で管理
- 一覧画面内の走り書きメモと録音
- 追加ボタンからタスク、メモ、運営情報を選択して追加
- タスクごとの担当者設定
- 実施日と `DL` の分離、共通カレンダーでの日付選択
- 任意の開始/終了時間の折りたたみ入力
- 実施日選択時の優先度空き状況表示
- 週次、月次日付、月次曜日、月初、月末、カスタム日付の繰り返しタスク
- タスク一覧の実施日順ソート
- タスク削除前の確認
- 削除済みタスク、メモ、運営情報の保管と復元
- 優先度枠の重複時に、既存タスクの移動先を選べる入れ替え UI
- 月曜始まりカレンダー、分類フィルター、DL/運営情報の簡易表示
- メモタイトルと本文プレビューからの編集、タスク側からの関連メモ確認
- AI整理用JSONファイルの共有と、AI整理結果JSONの取り込み
- メモ本文、議題、方針、行動の入力欄拡張
- メモ編集画面の関連タスク開閉
- 録音メモの端末内保存と専用欄への文字起こし保存
- タスク側からの関連メモ検索、運営情報の種別その場追加
- 設定画面での分類・種別のドラッグ並び替え
- JSON / CSV / テキスト貼り付けインポート
- JSON エクスポート
- IndexedDB によるローカル保存

## 小型サーバー準備

ローカル開発サーバーには将来拡張用の確認 API を追加しています。

- `GET /api/health`: サーバー起動確認
- `GET /api/capabilities`: 常時同期、AI秘書化、音声常駐、Apple Watch、バックグラウンド自動処理の予定状態

## 公開URLとマスタデータ

GitHub Pages の公開URLは静的配信なので、アプリから安全にマスタファイルへ直接書き戻すことはしません。直接書き込みを公開ページに持たせると、認証情報や書き込み口の扱いが難しくなります。

当面の運用は、設定の「マスター上書き用JSON」でiPhone側の最新状態を書き出し、PC側で `data/claris-master-2026-05-18.json` の更新に使う形です。アプリ起動時にこのJSONを取得し、`fullSync: true` かつ未適用の `importId` であれば、この端末のIndexedDBへ上書き同期します。パスキーなどで安全に双方向同期する場合は、GitHub Pagesだけで完結させず、認証と書き込み権限を持つバックエンドを別途用意します。
