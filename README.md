# Claris / クラリス

自分用のタスク、メモ、方針管理アプリです。まずは iPhone の Safari でホーム画面に追加して使える、ローカル保存の PWA として実装しています。

## 起動

```powershell
node server.mjs
```

起動後に `http://127.0.0.1:4173/` を開きます。

## 主な機能

- 下部タブ中央の「今日」
- タスク、メモ、方針、カレンダーの管理
- 任意の作業者名設定
- 実施日と `DL` の分離
- 優先度枠の重複時の入れ替え UI
- 走り書きメモから議題、決定事項、ネクストアクションへの整理
- 録音メモの端末内保存
- JSON / CSV / テキスト貼り付けインポート
- JSON エクスポート
- IndexedDB によるローカル保存
