# レビュー反映メモ 2026-05-15

## 反映した判断

- アプリ名は `Claris / クラリス` に変更した。
- 下部タブは奇数の 5 個にし、中央に「今日」を置いた。設定は下部タブから外し、右上メニューへ移した。
- 作業者名は任意設定にした。今日画面のヘッダーに担当者として表示する。
- 期限日は画面表記を `DL` に統一した。実施日は `実施` として残す。
- メモは走り書き保存を優先し、後から「自動判定」で論点、方針、行動へ展開できるようにした。
- 録音は端末内保存のメモ添付として実装した。文字起こしはブラウザの音声認識が使える場合だけ補助機能として動く。
- ChatGPT からの書き込みは、読み込みなしの貼り付けインポートとして受けられる形にした。
- 優先度枠の重複時は、警告だけでなく、既存タスクをサブタスク化、翌日移動、入れ替え、同枠追加から選べる。
- PWA は採用した。iPhone ホーム画面利用とオフライン利用のために、manifest と service worker を用意した。

## まだ意図的に軽くしている部分

- 文字起こしは完全ローカル AI ではない。外部 API 連携は入れていない。
- Microsoft To Do 連携は API ではなく、テキスト貼り付けインポートで受ける。
- 複数端末同期、読み込み型 GPT 連携、クラウド保存は未実装。
